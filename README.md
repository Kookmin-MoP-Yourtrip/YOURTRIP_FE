###✨ 프로젝트 소개

YOURTRIP은 사용자가 여행을 계획할 때 겪는
⦁ 정보 과다
⦁ 일정 조율의 어려움
⦁ 목적지 탐색 피로도

이런 문제들을 해결하기 위해 제작된 여행 코스 플래너 앱입니다.

AI 기반 추천 기능을 포함해, 직접 코스를 만들고 관리할 수 있으며
서로의 여행 기록을 공유할 수 있습니다. 
또한 마음에 드는 다른 사람의 코스를 fork 해와서 편하게 수정 가능합니다
친구와 함께 실시간으로 코스를 작성하며 여행 계획을 짜보실 수도 있습니다
