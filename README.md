### ✨ 프로젝트 소개

YOURTRIP은 사용자가 여행을 계획할 때 겪는
- 정보 과다
- 일정 조율의 어려움
- 목적지 탐색 피로도

이런 문제들을 해결하기 위해 제작된 여행 코스 플래너 앱입니다.

YOURTRIP은 AI 기반 추천 코스를 제공할 뿐만 아니라,
직접 코스를 생성하고 관리할 수 있는 기능을 제공합니다.

또한 사용자 간 여행 코스를 공유하고 fork하여,
마음에 드는 코스를 가져와 나만의 여행 계획으로 재구성할 수 있습니다.

친구와 함께 실시간 공동 편집이 가능해
여행 계획을 더 쉽고, 더 즐겁게 만들 수 있는 것을 목표로 합니다.
